Decisions (v0)
Menu bar lives only under Surfaces
No duplicate quick-link group — keeps the rail clean
Custom rules stay in Notifications for now
They graduate to an Automations pane once actions go beyond Notify
Usage choices default per integration
New instances inherit; each instance can override; "Apply to all" offered
Combined vs separate = per-surface default + per-item override
One switch sets the default; any item can still be split or merged
